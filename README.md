# Deep_learning_systems_2026

Репозиторий демонстрирует запуск модели Bolmo (allenai/Bolmo-1B) в контейнере Docker.

## О модели
Bolmo: Byte-level Language Models
Bolmo — это семейство полностью открытых байтовых LLM, полученных путём байтификации существующих моделей Olmo (1B и 7B). В отличие от большинства LLM, которые работают с субсловными токенами, Bolmo обрабатывает текст на уровне отдельных UTF-8 байтов. Это даёт ряд преимуществ:

- Полное понимание отдельных символов (нет потери информации на границах токенов).
- Устойчивость к токенизационным смещениям (например, проблема завершения промпта в середине слова).
- Лучшая работа с кодом, биологическими последовательностями и другими научными данными, где важен каждый символ.
- Возможность гибко изменять степень сжатия байтов в патчи, ускоряя инференс без изменения модели.

## Требования
- Наиличие видеокарты nvidia
- Git
- Docker
- Опционально: Hugging Face token (если модель требует авторизации или файлы очень большие)

## Запуск

1) Клонировать репозиторий проекта:

```bash
git clone https://github.com/Jopepka/Deep_learning_systems_2026.git
cd Deep_learning_systems_2026
```
2) Установитье hugging-face-токен для быстрой загрузки модели

Вставьте hugging-face токен в docker-compose

``` 
    environment:
      - HUGGING_FACE_TOKEN="Токен hugging-face" 
```

3) Запустить Docker Compose


```bash
docker-compose up --build
```
